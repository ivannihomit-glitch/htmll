# FitBalance — умный спортивный трекер для похудения

## Описание
FitBalance — это интерактивное веб-приложение для расчёта индивидуального коридора калорий на основе формулы Mifflin-St Jeor. Приложение учитывает пол, возраст, вес, рост, уровень физической активности и желаемый темп похудения, после чего предлагает персонализированные рекомендации по типу тренировок и упражнениям.

## Функциональность
- 📊 Расчёт базового обмена веществ (BMR) и суточного расхода калорий (TDEE)
- 🎯 Автоматический подбор калорийности для похудения с учётом безопасного минимума
- 🏋️ Индивидуальные рекомендации по типу тренировок (кардио, силовые, комбинированные)
- 📝 Детальные советы по упражнениям в зависимости от ИМТ, пола и возраста
- 🔄 Мгновенное автообновление результатов при изменении параметров
- 📱 Полностью адаптивный дизайн для всех устройств

## Технологии
- HTML5
- CSS3
- JavaScript (ES6)
- Формула Mifflin-St Jeor
